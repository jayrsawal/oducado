# Oducado Family Reunion 2026

A Great Gatsby-themed website for the Oducado Family Reunion.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The site will open automatically at http://localhost:3000

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- CSS3 with Art Deco styling

## Features

- Great Gatsby/Art Deco theme
- Responsive design
- Animated sparkle effects
- Fade-in animations
- Interactive hover effects
